## Что это за проект

`workflow-planner` — скил Claude Code. Берёт задачу, решает, нужна ли ей параллельная
оркестрация (Dynamic Workflow), и пишет план: линейный (`PLAN.md`) или с ветками + готовый
JS-скрипт для Workflow tool. Внутри репо есть вторая копия под Codex
(`plugins/workflow-planner-codex/`) — та же логика, другой рантайм.
